# PE-IoT PET Manager

## Software Requirements

**Supported OS**: Linux / macOS 

**Required Softwares**:

- Bash
- Gradle: (version: 6.0+)
- Java (JDK): (version: 11+)

## Program Execution
- `gradle build`
- `java -jar build/libs/peiot-petmanager-0.0.1-SNAPSHOT.jar` (the JAR filename may change according to the configurations in `build.gradle` and `settings.gradle` )
